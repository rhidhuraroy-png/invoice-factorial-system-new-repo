# 🧾 Permissionless Invoice Factoring DApp (Soroban)

## 📌 Project Description

This project is a **fully permissionless Invoice Factoring DApp** built on **Stellar Soroban**.

Unlike traditional systems, there is:

* ❌ No admin
* ❌ No approval system
* ❌ No centralized control

Anyone can interact with the protocol freely, making it aligned with **DeFi principles**.

---

## ⚙️ What It Does

* Allows anyone to create invoices
* Enables open participation in invoice funding (factoring)
* Lets anyone trigger invoice settlement
* Stores all data transparently on-chain

---

## ✨ Features

* 🧾 **Permissionless Invoice Creation**
* 💰 **Open Funding (Factoring)**
* 🔄 **Decentralized Settlement**
* 🔍 **Transparent On-chain Data**uo
* ⚡ **No Central Authority**

---

## 🔁 Workflow

1. Seller creates invoice
2. Any user funds it (becomes factor)
3. Buyer repays invoice
4. System updates status
